# Use Case Descriptions (Step-by-Step)

### 1. 회원가입
**관련 요구사항:** FR-01

| Actor Action (비로그인 사용자) | System Response |
| :--- | :--- |
| | 1. ID, 비번, 이름, 전번, 이메일 적는 칸을 보여준다. |
| 2. 빈칸 다 채우고 가입 버튼을 누른다. | |
| | 3. 가입 처리를 완료하고 성공 메시지를 띄운다. |

- **Alternative Courses** 
step 2-3. ID가 중복이거나 필수 정보가 누락되면 에러를 띄우고 기존 화면을 유지한다.

---

### 2. 로그인
**관련 요구사항:** FR-02

| Actor Action (관리자, 회원) | System Response |
| :--- | :--- |
| | 1. ID랑 비번 입력하는 로그인 창을 보여준다. |
| 2. 내 정보 입력하고 로그인 버튼을 누른다. | |
| | 3. 인증을 완료하고 권한(관리자/회원)에 맞는 메인 화면으로 보낸다. |

- **Alternative Courses** 
step 2-3. ID, 비번이 틀리면 로그인 실패 메시지를 알리고 다시 입력하게 함.

---

### 3. 설정 화면 (로그아웃, 탈퇴)
**관련 요구사항:** FR-03, FR-04

| Actor Action (관리자, 회원) | System Response |
| :--- | :--- |
| | 1. 로그아웃이랑 회원 탈퇴 버튼이 있는 설정 화면을 보여준다. |
| 2. 로그아웃이나 탈퇴 버튼 중 하나를 누른다. | |
| | 3. 요청한 작업(로그아웃 또는 탈퇴)을 처리하고 초기 화면으로 보낸다. |

---

### 4. 설문 등록
**관련 요구사항:** FR-05

| Actor Action (관리자) | System Response |
| :--- | :--- |
| | 1. 제목, 설명, 문항, 날짜 등 적을 수 있는 등록 화면을 보여준다. |
| 2. 설문 내용 다 채워넣고 등록 버튼을 누른다. | |
| | 3. 등록 처리를 완료하고 설문 목록 화면으로 이동한다. |

---

### 5. 설문 목록 조회 및 관리
**관련 요구사항:** FR-06, FR-07, FR-08

| Actor Action (관리자) | System Response |
| :--- | :--- |
| | 1. 지금까지 등록된 전체 설문 리스트를 보여준다. |
| 2. 특정 설문을 클릭한다. | |
| | 3. 팝업으로 상세 내용을 보여준다. |
| 4. 특정 설문을 선택해 삭제 버튼을 누른다. | |
| | 5. 해당 항목을 삭제 처리하고 목록을 새로고침하여 출력한다. |

---

### 6. 설문 검색
**관련 요구사항:** FR-09

| Actor Action (회원) | System Response |
| :--- | :--- |
| | 1. 설문 상태 필터랑 키워드 입력창을 보여준다. |
| 2. 상태 고르고 키워드 쳐서 검색을 누른다. | |
| | 3. 조건에 맞는 설문들만 리스트로 출력한다. |

---

### 7. 설문 상세 정보 확인
**관련 요구사항:** FR-10

| Actor Action (회원) | System Response |
| :--- | :--- |
| | 1. 선택한 설문의 상세 정보랑 선택 버튼을 보여준다. |
| 2. 진행 중인 설문이면 선택 버튼을 누른다. | |
| | 3. 실제 응답할 수 있는 화면으로 넘겨준다. |

---

### 8. 설문 응답
**관련 요구사항:** FR-11

| Actor Action (회원) | System Response |
| :--- | :--- |
| | 1. 설문 문항이랑 고를 수 있는 선택지들을 보여준다. |
| 2. 각 문항에 답 체크하고 응답 제출 버튼을 누른다. | |
| | 3. 응답 제출을 완료하고 성공 메시지를 띄운다. |

---

### 9. 응답한 설문 목록 조회
**관련 요구사항:** FR-12

| Actor Action (회원) | System Response |
| :--- | :--- |
| | 1. 내가 참여했던 설문 리스트(제목, 응답내용, 날짜)를 보여준다. |
| 2. 특정 내역을 선택해서 누른다. | |
| | 3. 당시 내가 제출했던 상세 응답 내용을 보여준다. |

---

### 10. 응답 수정 및 취소
**관련 요구사항:** FR-13, FR-14

| Actor Action (회원) | System Response |
| :--- | :--- |
| | 1. 기존 응답이 채워진 수정 폼이랑 취소 버튼을 보여준다. |
| 2. 내용을 고쳐서 제출하거나, 응답 취소 버튼을 누른다. | |
| | 3. 요청에 맞춰 변경 사항을 반영하거나 취소 처리 후 결과를 보여준다. |

- **Alternative Courses** 
step 1. 설문 기간이 끝났으면 수정이나 취소 버튼을 비활성화해서 안 보여줌.

---

### 11. 통계 조회
**관련 요구사항:** FR-15

| Actor Action (관리자) | System Response |
| :--- | :--- |
| | 1. 기간(1주/1달/1년)을 선택할 수 있는 통계 화면을 보여준다. |
| 2. 확인하고 싶은 기간 단위를 고른다. | |
| | 3. 전체 설문 수, 종료된 설문 수, 총 응답 수를 산출하여 화면에 출력한다. |
